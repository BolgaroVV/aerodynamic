Модель атмосферы, представленная в виде распределения средних значений параметров по высоте
Параметры международной стандартной атмосферы равны средним значениям параметров реальной атмосферы на средних широтах в летнее время
Параметры стандартной атмосферы для нулевого уровня отмечают индексом "с":
- $T_c = 288.15K$
- $p_c$ = 101300 Па
- $\rho_c$ = 1.225 кг/$м^3$ 
- $a_c = 340.29 м/с$
- $\nu_c = 1.46*10^-5 м^2/с$
